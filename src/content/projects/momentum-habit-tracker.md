## Overview

Momentum is a focused habit and routine tracker that helps users build streaks, visualise progress, and keep momentum with reminders and analytics.

## Key Features

- Habit creation, recurring schedules and configurable reminders.
- Streak tracking and history charts using Chart.js for quick insights.
- Clean, distraction-free UI designed for habit consistency.
- Secure user accounts and JWT-based authentication for the API.

## Architecture & Tech

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Charts: Chart.js

## Run locally

```bash
git clone https://github.com/pri2003yam/Momentum.git
cd Momentum
npm install
# create .env and run server + client
npm run dev
```
